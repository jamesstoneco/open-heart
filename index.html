Why use Open Heart?

jQuery Love

Open Heart is designed to work with all the jQuery goodness from the ground up. Love jQuery? You'll love Open Heart.

Open Source

MIT License. Thats right, go crazy. Make awesome stuff. Put a fork in it. Create some commercial goodness without worries or regrets.

Share the Love

Anything that you can create, we think will be better with some Heart. Share your work and get recoginzed in the Open Heart community.

Focused

Tired of kitchen sink libraries? Don't want vampires and zombies messing up your Heart(s)? We stay focused on creating a library that does only one thing and does it at best of class level.

Security

Is your Heart not in it? Worried about hackers? Open Heart sits client side on top of jQuery so it is as secure as jQuery is.

Heart Healthy?

It seems that Hearts are good for the Heart. Make people happy. Share the love. Valentines Day should be Valentines Year.

The Basics

Get Started in 60 Seconds or Less

Step 1

Add jQuery and link the library

<script src="javascripts/vendor/jquery.js"></script>
<script src="javascripts/open-heart/openheart.js"></script>
Step 2

Add the following to your CSS, SCSS or Sass file:

.heart { 
  width: 126px; 
  height: 117px; 
  position: fixed; 
  background-image: url('../javascripts/open-heart/heart.png'); 
  z-index: 99999;
}
Step 3

Init Open Heart

javascript

hearts(10);

coffeescript

hearts 10

Starting and Stoping

One of the easiest ways to start and stop the hearts is by binding to click events using jQuery.

javascript

$(document).ready(function() {
  $('#start-hearts').click(function() {
    hearts(10);
  });
  $('#kill-all-hearts').click(function() {
    kill_all_hearts();
  });
});

coffeescript

$(document).ready ->
  $('#start-hearts').click ->
    hearts(10)
  $('#kill-all-hearts').click ->
    kill_all_hearts()

You are not limited to click events. You can call hearts() and kill_all_hearts() at anytime and link them to any event. Want your hearts only for 30 seconds? No problem, just set up a timer and call kill_all_hearts() to remove them.

Basic CSS Customization

Retina Hearts

If you would like to have all the retina goodness and don't mind having a performance hit on older browsers, you can change the following line to use the @2x png.

background-image: url('../javascripts/open-heart/heart@2x.png');
Custom Hearts

If you would like to use your own custom hearts, go crazy. Just remember to adjust the height and width parameters to match the size of your hearts.

Hidden Hearts

If you would like to have your hearts appear behind your other HTML elements just reduce the z-index level until you are happy with the result.
